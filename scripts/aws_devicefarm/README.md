# AWS Device Farm Scripts

These scripts are Phase 0 stubs. They validate that `aws` is available, print usage, and show intended Device Farm commands. They do not include credentials and do not schedule real runs unless extended in a later phase.

Required later setup:

- AWS CLI configured by the user.
- `AWS_REGION`, usually a Device Farm-supported region.
- Device Farm project ARN.
- Device pool ARN.
- Uploaded app and test package ARNs.
