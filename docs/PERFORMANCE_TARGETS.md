# Performance Targets

The long-term trial target is at least 20 decode tokens/sec for the configurable Qwen-style 9B decoder model on the target Android phone class.

Current status:

- Workflow preparation only.
- No full model execution exists.
- No on-device benchmark has run.
- No performance number has been measured or claimed.

Future claims must include:

- Device model and SoC details from the probe.
- Backend used.
- Model metadata and quantization format.
- Context length and decode length.
- Thermal state.
- Raw benchmark JSON artifacts.
